# NM-2012-003

| Field | Value |
|-------|-------|
| **Date** | July 23, 2012 |
| **Location** | Lake of the Ozarks MO |
| **Classification** | Near Miss |
| **Outcome** | Dead: 0, Injured: 0, Shocked/survived: 3 |
| **Source Section** | Near Misses |
| **PDF Item Number(s)** | NM #40 |

## Description

Two children and an adult near a dock under electrical repairs at Lake of the Ozarks, Missouri all felt electric shock. Power was turned off and everyone exited safely.

## Classification Notes

Dock was under electrical repairs when incident occurred. Power shutoff prevented fatalities.
