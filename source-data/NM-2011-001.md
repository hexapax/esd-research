## Description

A dog was electrocuted in Hurricane Irene floodwater near Shady Beach, Norwalk, Connecticut. Two women felt shocks. Witnesses reported the "water was boiling."

## Classification Notes

Hurricane Irene context. Dog killed. "Water boiling" description suggests significant current flow. Outcome: Dead: 0 (human), Shocked/survived: 2.
