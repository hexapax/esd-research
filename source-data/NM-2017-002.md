# NM-2017-002

| Field | Value |
|-------|-------|
| **Date** | July 3, 2017 |
| **Location** | Lake Keowee SC |
| **Classification** | Near Miss |
| **Outcome** | Dead: 0, Injured: 0, Shocked/survived: 1 |

| **PDF Item Number(s)** | NM #18 |

## Description

A man's son grabbed the crossmember of a boat lift while under a boat at Lake Keowee, South Carolina. He was shocked but recovered. The boat lift was not properly grounded.

## Classification Notes

Boat lift not properly grounded. Child recovered.
