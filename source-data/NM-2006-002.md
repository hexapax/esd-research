# NM-2006-002

| Field | Value |
|-------|-------|
| **Date** | July 2, 2006 |
| **Location** | Lake L'Homme Dieu, Douglas County MN |
| **Classification** | Near Miss |
| **Outcome** | Dead: 0, Injured: 1, Shocked/survived: 3 |
| **Source Section** | Near Misses |
| **PDF Item Number(s)** | NM #59 |

## Description

Three men in a boat at Lake L'Homme Dieu, Douglas County, Minnesota had their prop catch an extension cord powering a boat lift. They were severely shocked upon entering the water. One was face down in the boat for several days in the hospital.

## Classification Notes

Extension cord caught by propeller. One victim hospitalized for several days. All survived.
