# NM-2014-002

| Field | Value |
|-------|-------|
| **Date** | July 4, 2014 |
| **Location** | Lake of the Ozarks MO, 7 mile marker |
| **Classification** | Near Miss |
| **Outcome** | Dead: 0, Injured: 0, Shocked/survived: multiple |
| **Source Section** | Near Misses |
| **PDF Item Number(s)** | NM #33 |

## Description

People were feeling tingles at the 7 mile marker on Lake of the Ozarks, Missouri. Turning off dock power didn't help. The source was a short 100 yards away at an abandoned boat ramp.

## Classification Notes

Remote source (100 yards away). Abandoned boat ramp as source. Dock power disconnect did not resolve issue.
